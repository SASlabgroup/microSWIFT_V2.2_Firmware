****************************************************************************************************
Iridium
*******

[x] Enabled reception of sbd messages. If message length matches expected length of configuration
    struct, the message contents will be applied as the new configuration. If a system error occurs,
    the current configuration will be discarded and overwritten with the original configuration
    saved in Flash memory. A type 99 SBD message is sent to acknowledge the new received config.
